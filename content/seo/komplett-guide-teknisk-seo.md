# Komplett guide til teknisk SEO

Teknisk SEO er fundamentet som alt annet bygger på – uten det kan Google ikke lese, forstå og indeksere sidene dine, uansett hvor godt innholdet er. Denne guiden dekker alt fra crawling til Core Web Vitals.

> **Kort forklart**
> Teknisk SEO handler om tre ting: at Google kan FINNE sidene dine (crawling), at Google kan FORSTÅ dem (rendering, strukturerte data), og at de er RASKE og brukervennlige (Core Web Vitals, mobil). Uten teknisk fundament er innhold og lenker verdiløse – Google ser dem aldri. Denne guiden og de lenkede artiklene gir deg alt du trenger.

## Teknisk SEO-sjekkliste

Den raske sjekklisten for teknisk SEO. Crawling: robots.txt tillater viktige sider, XML sitemap innsendt i Search Console, ingen viktige sider blokkert. Indeksering: viktige sider er indeksert (sjekk i Search Console), canonical tags på duplikater, hreflang for flerspråklig. Hastighet: Core Web Vitals i grønn sone, bilder komprimert og lazy-loaded, CDN for global hastighet. Struktur: HTTPS aktivert, mobilvennlig design, rene URL-er, intern lenking med logisk hierarki. Strukturerte data: schema markup for artikler, FAQ, produkter, og rich snippets validert i Rich Results Test.

## Oppsummering

Teknisk SEO er fundamentet – uten det ser Google ikke innholdet ditt. Crawling, indeksering, hastighet, schema og sidestruktur er kjerneelementene. Bruk Google Search Console som daglig verktøy. Screaming Frog for teknisk crawl. PageSpeed Insights for hastighet. Fiks tekniske problemer FØR du fokuserer på innhold og lenker – fundamentet må være på plass.
