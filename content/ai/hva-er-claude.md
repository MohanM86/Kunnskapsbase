# Hva er Claude?

Claude er en AI-assistent utviklet av Anthropic, et AI-sikkerhetsselskap grunnlagt i 2021 av tidligere OpenAI-forskere. Claude er designet med et spesielt fokus på å være hjelpsom, ærlig og ufarlig – et prinsipp Anthropic kaller «Constitutional AI». Claude er kjent for sterke resonneringsevner, svært lange kontekstvinduer og en gjennomtenkt tilnærming til AI-sikkerhet.

> **Kort forklart**
> Claude er en AI-assistent fra Anthropic som du kan snakke med via claude.ai, apper eller API. Den er kjent for å gi grundige, nyanserte svar og kan håndtere ekstremt lange dokumenter og komplekse oppgaver.

## Anthropic og AI-sikkerhet

Anthropic ble grunnlagt av Dario og Daniela Amodei sammen med andre tidligere OpenAI-forskere med en visjon om å bygge AI som er fundamentalt trygg. Selskapet bruker en tilnærming kalt Constitutional AI der modellen trenes med et sett etiske prinsipper som den internaliserer, i stedet for å stole utelukkende på menneskelig feedback.

Denne filosofien gjenspeiler seg i Claude – modellen er designet for å være ærlig om sin usikkerhet, unngå å generere skadelig innhold, og opprettholde en gjennomtenkt og nyansert tilnærming til kontroversielle temaer.

## Claude-modellene

Anthropic tilbyr flere modellvarianter tilpasset ulike behov.

### Claude Opus

Den mest avanserte og intelligente modellen. Opus er designet for de mest krevende oppgavene som kompleks analyse, avansert koding, akademisk forskning og oppgaver som krever dyp forståelse og kreativitet. Den er tregere og dyrere enn de andre modellene, men gir best kvalitet.

### Claude Sonnet

Mellommodellen som balanserer intelligens, hastighet og kostnad. Sonnet er det mest populære valget for de fleste oppgaver – den er rask nok for interaktiv bruk og smart nok for avanserte oppgaver. For de fleste brukere er Sonnet det beste utgangspunktet.

### Claude Haiku

Den raskeste og rimeligste modellen. Haiku er ideell for oppgaver som krever rask responstid med god kvalitet – som chatboter, klassifisering og enkel tekstbehandling. Den er vesentlig raskere enn Sonnet og Opus.

## Hva skiller Claude fra andre AI-assistenter

### Langt kontekstvindu

Claude kan håndtere opptil 200 000 tokens i kontekstvinduet – tilsvarende en hel bok. Dette betyr at du kan laste inn hele dokumenter, kodebaser eller datasett og stille spørsmål om innholdet. For mange brukere er dette den viktigste differensiatoren sammenlignet med andre AI-assistenter.

### Grundig resonnering

Claude er kjent for å gi grundige, velstrukturerte svar som vurderer flere perspektiver. I stedet for raske, overfladiske svar tar Claude seg tid til å analysere spørsmålet og gi en nyansert respons.

### Ærlighet om usikkerhet

Claude er trent til å si «jeg vet ikke» eller «jeg er usikker» når den mangler kunnskap, i stedet for å hallusinere et plausibelt svar. Denne ærligheten gjør Claude mer pålitelig som informasjonskilde.

### Verktøybruk og MCP

Claude støtter verktøybruk direkte – den kan søke på nettet, lese filer, kjøre kode og koble seg til eksterne tjenester via MCP (Model Context Protocol). Claude Code er Anthropics kommandolinjeverktøy der Claude opererer som en kodingsagent som kan skrive, redigere og kjøre kode autonomt.

## Tilgangsmåter

### claude.ai

Webgrensesnittet claude.ai gir direkte tilgang til Claude via nettleseren. Gratisplanen gir begrenset bruk av Sonnet. Pro-abonnementet gir tilgang til alle modeller, høyere bruksgrenser og tilleggsfunksjoner.

### Claude-appen

Native apper for macOS, iOS og Android gir tilgang til Claude med samme funksjonalitet som webversjonen, pluss integrasjon med operativsystemets funksjoner.

### API

Anthropics API gir utviklere tilgang til Claude-modellene for å bygge applikasjoner. API-et støtter tekstgenerering, bildAnalyse, verktøybruk og streaming, og er tilgjengelig via Python og TypeScript SDK-er.

### Claude Code

Claude Code er et kommandolinjeverktøy der Claude fungerer som en agentisk kodingsassistent. Den kan navigere og forstå kodebaser, skrive og redigere filer, kjøre kommandoer og tester, og iterere basert på feilmeldinger.

## Bruksområder

### Skriving og analyse

Claude er sterk på lange, grundige tekster – artikler, rapporter, analyser og sammendrag. Det lange kontekstvinduet gjør den ideell for å analysere store dokumenter og syntetisere informasjon fra mange kilder.

### Koding

Claude er en av de sterkeste AI-modellene for programmering. Den kan skrive, forklare og debugge kode i de fleste språk, og Claude Code gjør den til en fullverdig kodingsagent.

### Research

Med evnen til å prosessere lange dokumenter kan Claude lese forskningsartikler, teknisk dokumentasjon og juridiske tekster, og gi presise sammendrag og analyser.

### Kreativt arbeid

Claude kan skrive kreative tekster, brainstorme idéer, utvikle konsepter og gi tilbakemelding på kreativt arbeid – med en gjennomtenkt tilnærming som vurderer flere vinkler.

## Claude og norsk språk

Claude forstår og genererer norsk bokmål godt og er blant de sterkeste AI-modellene for norsk. Den kan skrive profesjonelt norsk med riktig grammatikk og idiomatiske uttrykk, forstår norsk kulturell kontekst og samfunnsforhold, kan håndtere norsk fagterminologi innen de fleste felt, og kan skille mellom bokmål og nynorsk.

For norske bedrifter og brukere gir Claude en pålitelig norskspråklig AI-assistent med kvalitet som nærmer seg engelskspråklig ytelse.

## Personvern og sikkerhet

Anthropic tar personvern seriøst. Samtaler i claude.ai brukes ikke til å trene modeller med mindre du eksplisitt samtykker. API-bruk er helt adskilt fra trening. For bedrifter tilbyr Anthropic enterprise-avtaler med databehandlingsgarantier.

Claude har ingen annonser og anbefaler ikke produkter eller tjenester. Samtaledata deles ikke med tredjeparter.

## Ofte stilte spørsmål

### Er Claude gratis?
Ja, claude.ai har en gratisplan med begrenset bruk. Pro-abonnementet koster ca. 200 kr per måned og gir høyere bruksgrenser og tilgang til alle modeller.

### Hva er bedre – Claude eller ChatGPT?
Det avhenger av oppgaven. Claude er generelt sterkere på lange dokumenter, grundig analyse og koding. ChatGPT har et bredere økosystem med plugins og GPTs. Mange bruker begge for ulike formål.

### Kan Claude brukes til bedriftformål?
Ja. Anthropic tilbyr API med dataskillingsgarantier, og claude.ai bruker ikke samtaledata til trening uten samtykke. For sensitive bedriftsdata er API-tilgang anbefalt.

### Kan Claude lese filer og bilder?
Ja. Claude kan analysere bilder, lese PDF-er og prosessere dokumenter. Via MCP og Claude Code kan den også interagere med filsystemer og eksterne tjenester.
