# Velge webhotell

Et webhotell er en tjeneste som lagrer nettstedet ditt på en server slik at det er tilgjengelig på internett.

> **Kort forklart**
> Webhotell er serverplass du leier. Delt hosting er billigst, VPS gir mer kontroll, og cloud-tjenester som Vercel er gratis for små prosjekter.

## Hva betyr begrepet

Hosting er serverplass for nettstedet ditt. Ulike typer passer for ulike behov og budsjetter.

## Hvordan fungerer det

```
Domene → DNS peker til webhotell → Server leverer nettstedet → Bruker ser siden
```

| Type | Pris/mnd | Passer for |
|------|----------|-----------|
| Delt hosting | 30–150 kr | Blogger, enkle sider |
| VPS | 80–500 kr | Voksende sider |
| WordPress | 150–800 kr | WordPress med ytelseskrav |
| Cloud | Gratis+ | Next.js, statiske sider |

## Hvorfor er det viktig

Hostingvalg påvirker hastighet, oppetid og sikkerhet direkte.

## Eksempler

| Bruk | Anbefaling |
|------|------------|
| Personlig nettside | One.com, Domeneshop |
| WordPress | WP Engine, Domeneshop |
| Next.js | Vercel (gratis) |
| Skalerbar app | Hetzner VPS |

## Vanlige spørsmål

### Delt hosting vs VPS?
Delt deler ressurser. VPS gir dedikerte ressurser.

### Er gratis hosting bra nok?
For hobbyprosjekter. For bedrifter: betalt.

### Trenger jeg norsk webhotell?
Ikke nødvendigvis. CDN løser hastighet.

### Hva er Vercel?
Cloud-plattform for Next.js. Gratis for små prosjekter.

### Inkluderer webhotell SSL?
De fleste inkluderer gratis SSL i dag.
