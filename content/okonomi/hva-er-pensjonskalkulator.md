# Hva er pensjonskalkulator?

En pensjonskalkulator beregner hva du kan forvente å få i pensjon – fra folketrygd, tjenestepensjon, AFP og egen sparing – basert på din inntekt, alder og opptjening.

> **Kort forklart**
> De fleste nordmenn aner ikke hva de får i pensjon. En pensjonskalkulator gir svaret: logg inn på nav.no/din-pensjon med BankID → se din folketrygd-prognose. Sjekk norskpensjon.no → se ALL pensjon samlet (folketrygd + tjenestepensjon + AFP). Resultatet er typisk en overraskelse – de fleste får MINDRE enn de tror. Det er det viktigste finansielle 10-minuttene du kan bruke: sjekk → forstå gapet → start å spare.

## De beste pensjonskalkulatorene

Norske kalkulatorer du bør bruke.

```
1. nav.no/din-pensjon (VIKTIGST)
   → Logg inn med BankID
   → Se din folketrygd-prognose ved ulike uttaksaldre
   → Basert på din FAKTISKE opptjening – mest nøyaktig

2. norskpensjon.no (SAMLET OVERSIKT)
   → Viser ALLE pensjonskilder: folketrygd + tjenestepensjon + AFP
   → Gir deg totalbilde

3. Bankenes kalkulatorer:
   DNB, Nordea, Storebrand, KLP – egne pensjonskalkulatorer
   → Kan inkludere egen sparing i beregningen
   
4. smartepenger.no/pensjonskalkulator
   → Uavhengig, pedagogisk, inkluderer alle søyler

Gjør dette NÅ:
  □ Logg inn på nav.no/din-pensjon → Se folketrygd-prognose
  □ Logg inn på norskpensjon.no → Se totalbilde
  □ Beregn gapet → Hva mangler for å opprettholde levestandarden?
  □ Start å spare → Dekk gapet med indeksfond via ASK
```

## Hva bør du sjekke

Når du bruker kalkulatoren, se på forventet årlig pensjon ved 62, 67 og 70 år, prosent av nåværende inntekt (erstatningsgrad), pensjonsgapet (hva du mangler for 80+ prosent av nåværende inntekt), og tjenestepensjonens bidrag (er 2 prosent OTP nok?).

## Vanlige spørsmål

### Er NAVs kalkulator nøyaktig?
For folketrygd: ja – basert på din faktiske opptjening. For tjenestepensjon: bruk norskpensjon.no i tillegg. For fremtidig: alt er estimat – inntekt, avkastning og regler kan endre seg.

### Hva om jeg er ung med lite opptjening?
Kalkulatoren projiserer basert på nåværende inntekt fremover. Jo yngre du er, jo mer usikkert – men den gir et nyttig estimat.

### Hvor ofte bør jeg sjekke?
Hvert 5. år minimum. Ved jobbbytte (ny pensjonsordning). Og når du nærmer deg 55–60 (planlegge uttak).

## Oppsummering

Pensjonskalkulator viser hva du får i pensjon. Bruk nav.no/din-pensjon (folketrygd) og norskpensjon.no (totalbilde). De fleste får MINDRE enn de tror. 10 minutter å sjekke – det viktigste finansielle du kan gjøre. Identifiser pensjonsgapet → start å spare med indeksfond via ASK. Sjekk hvert 5. år.
